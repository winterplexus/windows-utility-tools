
 Utility Tools - Event Log Tool
 ===============================================================================

 Synopsis
 -------------------------------------------------------------------------------

 Utility tool to create or delete a Windows event log.

 Executable
 -------------------------------------------------------------------------------

 EventLogTool.exe

 Command Line Options
 -------------------------------------------------------------------------------

 -c [create|delete]
 -n <event name>
 -l <event log name>

 where:

 [] contains options
 <> contains parameters (quoted string if parameter contains space characters)
